Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • ParágrafosUse a tag p
  • TítulosUse a tag h
  • ImagensUse a tag img
  • AbreviaçõesUse a tag abb
  • Códigos-fonteUse a tag code
  • Lista numeradasUse a tag ol
  • Lista com demarcadoesUse a tag ul
